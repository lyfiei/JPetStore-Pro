使用 Pinia 管理购物车状态：
- cartItems: 购物车商品列表
- cartCount: 商品总数
- cartTotal: 总金额
- 同步后端数据

### 🎯 核心功能流程

**购物流程**：
1. 浏览首页/分类页 → 查看商品
2. 商品详情页 → 选择规格和数量 → 加入购物车
3. 购物车 → 确认商品 → 去结算
4. 结算页 → 选择地址和支付方式 → 提交订单
5. 订单详情页 → 查看订单状态

**用户流程**：
1. 注册账号 → 登录
2. 个人中心 → 编辑资料
3. 管理收货地址
4. 查看订单历史
