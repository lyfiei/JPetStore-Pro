# JPetStore 前端快速启动指南

## 🚀 5分钟快速启动

### 前置要求

确保已安装：
- Node.js (推荐 v16+)
- npm 或 yarn

### 步骤 1: 安装依赖

```bash
cd F:\code\JPetStore\jpetstore-storefront
npm install
```

### 步骤 2: 启动开发服务器

```bash
npm run dev
```

### 步骤 3: 访问应用

打开浏览器访问：**http://localhost:5173**

---

## 🔧 后端API配置

默认API地址：`http://localhost:8080/api/v1`

如需修改，编辑 `src/api/request.js`：

```javascript
const request = axios.create({
  baseURL: 'http://localhost:8080/api/v1', // 修改这里
  timeout: 10000
})
```

## 🐛 常见问题

### 1. 页面空白

**原因**: 后端API未启动或连接失败

**解决**: 
- 确保后端服务运行在 `http://localhost:8080`
- 检查浏览器控制台是否有CORS错误

### 2. 登录失败

**原因**: 后端认证接口问题

**解决**:
- 检查后端日志
- 确认数据库中有测试数据
- 验证API路径是否正确

### 3. 图片不显示

**原因**: 图片资源不存在

**解决**:
- 这是正常的，系统使用占位图
- 可以添加真实图片到 `public/images/` 目录

### 4. 购物车无法添加

**原因**: 未登录或Token过期

**解决**:
- 先登录账号
- 清除localStorage后重新登录
