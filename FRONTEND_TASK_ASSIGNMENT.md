## 🐛 常见问题排查

### 1. API 调用失败
**检查**：
- 后端是否启动
- API 地址是否正确
- Token 是否有效
- 网络是否正常

**解决**：
```javascript
// 检查 request.js 中的 baseURL
baseURL: 'http://localhost:8080/api/v1'
```

### 2. 页面空白
**检查**：
- 控制台是否有错误
- 路由配置是否正确
- 组件是否正确导入

**解决**：
- 查看浏览器控制台
- 检查 vite 输出
- 重启开发服务器

### 3. Token 失效
**检查**：
- localStorage 中是否有 token
- Token 是否过期
- 后端 JWT 配置

**解决**：
- 重新登录
- 检查 JwtUtil 配置
- 清除缓存后重试

### 4. 样式错乱
**检查**：
- CSS 是否正确加载
- Element Plus 样式是否引入
- 是否有样式冲突

**解决**：
- 检查 main.js 中是否引入 Element Plus CSS
- 检查 scoped 样式
- 清除浏览器缓存
